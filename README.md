# README
# E-Commerce Sales Data Analysis

## 📌 Project Overview
The E-Commerce Sales Data Analysis project is developed using Python to analyze and visualize sales data from an online retail business. The project uses Pandas and Matplotlib libraries to extract insights and present them through charts and graphs.

## 🎯 Objectives
- Analyze E-Commerce sales data.
- Calculate total sales revenue.
- Perform product-wise sales analysis.
- Identify top-selling products.
- Visualize sales trends using charts.
- Generate meaningful business insights.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Microsoft Excel

## 📂 Dataset Information
The dataset contains 50 records with the following attributes:

- Order_ID
- Product_Name
- Category
- Quantity
- Price
- Sales_Amount
- Order_Date

## 📊 Features
- Dataset loading and preprocessing
- Total sales calculation
- Product-wise sales analysis
- Top-selling products identification
- Bar chart visualization
- Sales trend analysis
